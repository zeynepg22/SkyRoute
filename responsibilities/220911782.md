## Backend API Contributions

I implemented and integrated the main backend API endpoints of the platform.

### Course API Endpoints

| Endpoint | Purpose |
|---|---|
| `GET /courses/` | Returns all available courses |
| `GET /courses/{id}` | Returns a single course |
| `POST /courses/` | Creates a new course |
| `PUT /courses/{id}` | Updates course information |
| `DELETE /courses/{id}` | Deletes a course |

---

### Lesson API Endpoints

| Endpoint | Purpose |
|---|---|
| `GET /lessons/{id}` | Returns lesson details |
| `GET /lessons/course/{course_id}` | Returns lessons belonging to a course |
| `POST /lessons/` | Creates a new lesson |

---

### Enrollment API

| Endpoint | Purpose |
|---|---|
| `POST /enrollments/{course_id}` | Allows students to enroll in courses |

---

### Progress API

| Endpoint | Purpose |
|---|---|
| `POST /progress/complete/{lesson_id}` | Marks a lesson as completed |
| `GET /progress/{course_id}` | Returns student course completion percentage |
