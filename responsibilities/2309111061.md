# 2309111061 — Project Responsibilities

## Role

**QA Automation & DevOps Engineer**

This file explains my individual technical contributions to the **SkyRoute Online Course Platform** project.

---

## Main Responsibility Area

My main responsibility was establishing a robust testing infrastructure and automating the deployment pipeline. This includes backend API validation, frontend end-to-end (E2E) testing, and Continuous Integration (CI) setup to ensure software quality and stability.

---

## Technologies Used

| Area | Technology |
|---|---|
| Backend Testing | Pytest |
| Frontend Testing | Playwright |
| CI/CD Pipeline | GitHub Actions |
| Runtime Environment | Node.js & Python 3.13 |
| Browser Automation | Chromium (Headless) |
| API Simulation | TestClient (FastAPI) |

---

## Key Technical Contributions

* **Full-Stack Test Suite:** Developed automated tests for critical paths including user login, course enrollment, and unauthorized access prevention.
* **Automated CI Workflow:** Created a YAML-based GitHub Actions pipeline that triggers on every push to validate both backend and frontend integrity.
* **Dependency Mocking:** Implemented dependency overrides in FastAPI to simulate different user roles (Admin/Student) during testing.
* **Browser Automation:** Scripted complex UI interactions and visual assertions to verify the frontend dashboard's responsiveness and functionality.
